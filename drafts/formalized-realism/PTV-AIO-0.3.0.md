## 符号约定

- **Π** (Practropy / 实熵)
  - 定义：生产者燃烧的不确定性 (Uncertainty burned by producers)。
  - 单位：Bits。
  - 助记：生产侧的热力学成本。

- **Ψ** (Practhalpy / 实焓)
  - 定义：消费者减少的不确定性 (Uncertainty reduced by consumers)。
  - 单位：Bits。
  - 助记：消费侧的信息增益。

- **ΔΠ** (Delta Practropy / 实熵增量)
  - 定义：单次行为或干预所燃烧的不确定性增量。
  - 物理意义：局部过程的热力学损耗。

- **ΔΨ** (Delta Practhalpy / 实焓增量)
  - 定义：单次行为或干预所减少的不确定性增量。
  - 物理意义：局部过程的信息价值产出。

- **Π_ext** (Externalized Practropy / 外部化实熵)
  - 定义：由未获补偿第三方承担的不确定性增量。
  - 社会意义：负外部性 (Negative Externalities) 的量化。

- **Π_self** (Self-generated Practropy / 自生实熵)
  - 定义：平台为维持分发基础设施而燃烧的不确定性增量。
  - 系统意义：基础设施的运维熵增。

- **Γ** (Net Practhalpy / 净实焓)
  - 公式：Γ = ΔΨ − ΔΠ。
  - 定义：价值增益的最终判据 (Final criterion for value gain)。
  - 判据：仅当 Γ > 0 时，系统产生净正向价值。

- **Γ_ext** (Full-Scope Net Practhalpy / 全口径净实焓)
  - 公式：Γ_ext = ΔΨ − (ΔΠ + ΔΠ_ext)。
  - 定义：扣除外部化实熵后的社会净收益 (Social net benefit after externalities)。
  - 判据：衡量真实社会福利的唯一有效指标。

- **η** (Practropy Exchange Rate / 实熵汇率)
  - 公式：η = Ψ / Π。
  - 定义：系统效率指标 (System efficiency metric)。
  - 意义：单位熵增所换取的信息减熵比率。

## 概述

**实熵价值论（Practropy Theory of Value, PTV）** 将价值锚定于可审计的物理过程：生产者在压缩数据过程中实际燃烧的不确定性（**Π**），以及消费者因不确定性减少而获得的信息增益（**Ψ**）。价值批判的全部工作，归结为审计 Γ = ΔΨ − ΔΠ 的符号与幅度，并由此客观的回答“剥削如何发生”。

**热力学有效利他主义（Thermodynamic Effective Altruism, TEA）** 是 PTV 的规范性延伸，回答“在物理定律约束下如何最大化善”：一个行为的“善”正比于它所引发的净实焓增。

## 直观映射：一个电商例子

**将产品打到全网最低价的成本 ≈ 生产者的实熵（Π）。** 商家将价格压到“全网最低”所燃烧的不确定性——供应链谈判中的信息不对称、物流优化的探索成本、价格战中的策略博弈——这些消耗以 bits 计量。每一次决策都需在不确定性中搜索，每一次搜索都是 Π 的累积。

**初始心理价到最终消费价 ≈ 消费者的实焓（Ψ）。** 消费者看到 199 元时，其先验预测分布（“大概 250–300”）收敛到精确锚点（“就是 199”）。先验分布与后验分布之间的 KL 散度，即 Ψ，以 bits 计量。

**平台的角色：截留熵差。** 平台以趋零的自生实熵 Π_self（固定代码的匹配算法），强制商家燃烧更多 Π 以获取流量，同时用“全网最低价”标签重塑消费者心理先验，截留二者之间的熵差。商家疲惫，消费者麻木，平台利润增长——这是认知剪刀差（Π − Ψ）的直观呈现。

**意义通胀的微观机制。** 反复暴露于“最低价”标签会下移消费者的先验分布，使得同等 Π 燃烧产生的 Ψ 递减。系统总 Π 持续增长，边际 Ψ 趋零，文明实熵汇率 η = Ψ/Π 崩溃。

## 背景：价值理论的两次危机

经济学与价值哲学的历史可概括为两次锚定失败。

### 第一次锚定：劳动价值论

从亚当·斯密、大卫·李嘉图到卡尔·马克思，劳动价值论将商品价值锚定于生产所需的人类劳动。马克思进一步将价值精炼为“社会必要劳动时间”——在现有社会正常生产条件下，制造某种使用价值所需要的平均劳动时间。这一锚点具有直观的道德力量：如果价值来自劳动，那么占有他人劳动产品而不付出等价劳动即构成剥削。

然而，劳动价值论遭遇了根本性的可计算性障碍。社会必要劳动时间在结构上等价于 Kolmogorov 复杂度——寻找生成给定输出的最短程序长度。Kolmogorov 复杂度已被严格证明为不可计算函数。“社会必要劳动时间”的精确值在原则上不存在可终止的算法。马克思以“社会平均”“正常条件”等统计概念迂回，但统计均值并非最小必要值——前者是经验估计，后者是理想极限。锚点本身是不可计算的幽灵，批判遂悬空。

### 第二次锚定：边际效用论

19 世纪 70 年代的边际革命将价值锚点从生产成本转向消费者的主观评价。价值被定义为最后一单位商品带来的边际效用，价格即价值的可观测表现。这一框架具有卓越的可计算性——价格是公开可验证的，效用函数可通过选择行为反推。

但边际效用论付出了本体论代价：它将价值完全消解为主观偏好，切断了价值与生产过程的物理联系。如果价值仅由偏好决定，那么“剥削”成为一个空洞概念——任何自愿交易在边际效用框架内都是价值增益。批评者只能诉诸道德直觉，无法从理论内部导出剥削的形式化定义。价值遂沦为虚无。

### 第三次重构：将价值锚定于可审计的物理过程

从FEP和IPWT出发，我们完成了对人类“主观性”的信息论锚定。将此扩展到经济学，用边际效用论的可观测性框架，承载劳动价值论的锚定性要求。从而通过信息论工具，将价值重新定义为可审计的物理量。

PTV 从中提取出 Π 与 Ψ 的区分：Π 对应生产者在压缩数据时实际燃烧的不确定性；Ψ 对应消费者从压缩结果中获得的不确定性减少。

## 核心概念

### 实熵（Practropy, Π）

Π 是一个认知系统在将输入数据压缩为具有泛化能力的预测模型过程中，实际燃烧的、不可收回的不确定性增量，以 bits 计量。

Π意味着生产者的**不确定性增加**。每一次探索死胡同、每一次尝试失败的压缩策略、每一次信息擦除——这些操作在信息论上都体现为熵的增加。生产者燃烧的不确定性，构成了价值的物质基础。

Π 消耗记录在物理世界的因果账本中：模型训练的浮点运算次数（FLOPs 可换算为信息处理的比特上限）、人力的注意力时长（人脑信息处理率约 10–50 bits/s）、探索过的策略分支数——这些是物理事实，不依赖形而上学背书。

在自由能原理中，系统最小化变分自由能的过程持续排放 Π。自由能的复杂度项对应模型结构的描述代价（组织 Π）；预测误差项对应当前模型解释新数据的额外代价（适配 Π）。任何学习过程都在燃烧 Π。

### 实焓（Practhalpy, Ψ）

Ψ 是消费者在解码或应用认知产出时，实际感知到的不确定性减少量，以 bits 计量，衡量该产出在多大程度上降低了消费者自身的变分自由能。

Ψ 依赖于消费者的先验知识——同一篇机器学习论文对 AI 研究员的 Ψ 远高于对随机路人的 Ψ。这是实焓的消费者相对性：信息增益必然依赖于接收端的解码能力。它不是价值主观主义的回归，而是对信息增益的精确描述：增益量由先验与后验的 KL 散度决定，先验是客观的起点参数。

Ψ 并非主观偏好的循环定义。给定消费者的有限状态空间，Ψ 原则上可通过行为改变量、参数化建模或决策质量变化来测量和估计。

在 PTV 框架中，未被任何人转化为 Ψ 的 Π，在价值论意义上属于比特空转（bit-idling）——不确定性被燃烧了，但未照亮任何人。Ψ 是 Π 的价值归宿：只有成功降低消费者预测误差的 Π 消耗才完成价值循环。但这不消解生产的因果地位：没有 Π，Ψ 便无从产生。

### 净实焓（Net Practhalpy, Γ）

**定义：**

Γ = ΔΨ − ΔΠ

Γ 是 PTV 的核心判据，以 bits 为单位。正值表示系统产生了净不确定性减少（价值增益），负值表示系统的 Π 燃烧超过了其 Ψ 产出（净价值耗散）。

### 认知剪刀差（Cognitive Scissors Gap）

**定义：**

CSG = Π − Ψ = −Γ

## 与熵价值论（ETV）的对接

熵价值论（An Entropy Theory of Value, ETV）由 Jing Chen 等人系统发展，其核心命题为：**经济价值是稀缺性的熵度量，价值函数具有对数形式。** ETV 建立了价值与信息熵之间的数学联系，证明了垄断、市场结构、稀缺性如何决定商品的经济价值。

PTV 与 ETV 的关系可表述为：

> **ETV 描述了价值如何产生（价值 = 熵）。PTV 描述了价值如何流动（剥削 = 熵差）。**

具体而言：

| 维度     | ETV                                  | PTV                                          |
| -------- | ------------------------------------ | -------------------------------------------- |
| 核心公式 | 价值 = f(稀缺性) = 对数形式的熵函数  | 净实焓 Γ = ΔΨ − ΔΠ                           |
| 关注焦点 | 价值的经济决定因素（垄断、市场规模） | 价值的伦理分配（剥削、认知租金）             |
| 计量单位 | 货币（价值度量）                     | 比特（信息度量），可通过货币代理换算         |
| 核心问题 | “商品值多少钱？”                     | “谁的不确定性被燃烧了？谁的不确定性减少了？” |

ETV 为 PTV 提供了价值起源的物理锚点：如果价值本质上是熵，那么 Π（生产者燃烧的不确定性）就是价值的原料，Ψ（消费者减少的不确定性）就是价值的实现。PTV 的 Γ = ΔΨ − ΔΠ 可直接在 ETV 的框架中被理解为**熵差的再分配**——剥削就是熵差的截留，公平就是熵差的合理回馈。

这一对接使 PTV 不再孤悬于经济学的边缘：它成为 ETV 在伦理与分配维度的自然延伸。

### 剥削的热力学定义

PTV 提供了一种独立于道德直觉的形式化定义：

> **剥削** = 一个系统或实体，在未燃烧对等 Π 的情况下，捕获由其他实体的 Π 燃烧所产生的 Ψ。

执行逻辑：

1. 生产者燃烧 Π（承担不确定性的认知代价）；
2. 该压缩过程产生可传递的 Ψ；
3. 平台通过垄断分发渠道，捕获 Ψ 的主要部分；
4. 平台的自生实熵 Π_self（固定代码、边际趋零的匹配算法）远小于其截留的 Ψ。

生产者为熵增账单买单，平台收获信息增益。差额即熵差欺诈（entropy gap fraud）。

## Thermodynamic Effective Altruism

### 从价值审计到伦理优化

PTV 锚定了价值，但未规定如何行动。TEA 在 PTV 的基础上增加一条规范性公理：

> **善的梯度**：一个行为或干预的伦理价值正比于它所引发的净实焓 Γ = ΔΨ − ΔΠ，并严格扣除外部化实熵 ΔΠ_ext。

**行为伦理价值密度（Ethical Density）：**

ℰ = Γ_ext / ΔΠ = (ΔΨ − (ΔΠ + ΔΠ_ext)) / ΔΠ

**干预的全口径净实焓：**

Γ_ext = Σ_i ΔΨ_i − (Π_direct + Π_ext)

其中 Π_ext 为外部化实熵——由未获补偿第三方承担的不确定性增量（环境污染、数据标注劳工的认知耗竭、供应链不确定性）。

## 理论后果

### 公共物品的价值核算

知识、代码、语言模型等产出的 Ψ 高度扩散（无数人受益），Π 消耗却高度集中（少数生产者承担）。边际效用论无法解释为何公共物品生产者应获补偿——若边际效用趋零，边际激励趋零。PTV 的回答是：只要 Π 被燃烧，就产生了价值债权，即使无债务人支付。债权可能因搭便车问题而悬空，但债权本身在物理上真实存在。这一结论为基础科研与开源基础设施提供了信息论依据，不诉诸任何道德直觉。

### 意义通胀：信息时代的核心危机

当系统燃烧的 Π 总量持续增长而边际 Ψ 趋近于零时，文明的实熵汇率 η = Ψ/Π 崩溃。表现包括：刷三小时短视频后不确定性减少量为零；读五年博士论文被引三次；AI 生成内容洪流淹没人类创作者，Ψ 锚点溶解。意义通胀是 PTV 的危机理论，解释了信息时代的基本悖论——信息越丰富，意义越稀薄。其根因不是人心浮躁，而是 Π 供给远超 Ψ 接收容量，系统层面的信息论失衡。

### 认知热力学的公正观

PTV 不指向传统的“劳动者拥有生产资料”，因为 Π 不来自特定生产关系，而来自任何形式的自由能最小化过程。它指向一种信息生态学秩序：最小化 Π 的无效燃烧，最大化 Π→Ψ 的转化效率。这要求认知基础设施（教育、科研、网络协议、AI 治理）被设计和审计为公共事业。论证不依赖道德激情，而依赖 MDL 的直接推论——私有化认知公共品将人为提高整个文明的描述长度，降低 η。

### 尺度任意性

ℰ 算子在任意尺度（单个神经元到整个文明）保持形式不变。TEA 不预设作用群体域 G，但强制任何伦理宣称必须显式声明其所用的 G——这将伦理争论从“什么是善”转化为可审计的问题：“谁的 Π 应被计入？谁的 Ψ 被排除在外？”

### 时间性与沉没成本谬误

百年无人查阅的孤本，其持续燃烧的 Π 是否属于沉没成本？PTV 的回答是：保护行为本身即持有认知期权（cognitive option）。期权费（premium）在期权未被行权时并非“浪费”——它每存在一天，就为整个文明对冲了“对该领域完全无知”的风险。期权价值的评估，在原则上可通过预测市场（prediction market）聚合分散信息来完成，而非由任何单一审计者独断。

## 与边际效用论的关系

PTV 与边际效用论**算法同伦，本体异构**。两者共享优化数学结构（效用函数 ↔ 预测精度函数），但 PTV 计算可审计的 ΔΨ（以 bits 计量），而非难以量化的主观偏好强度。偏好是 Ψ 的历史积分——PTV 直接锚定于更基底的自由能最小化过程，而非其历史积累的痕迹。

## 附录：FAQ

**Q1：实熵和实焓的单位为什么是 bits？**

A1：0.3.0 版本将 Π 重定义为生产者**增加**的不确定性，Ψ 为消费者**减少**的不确定性。二者同以 bits 计量，在量纲上完全对齐。这解决了早期版本中 Π 跨热力学与信息论双维度的“汇率问题”——焦耳与比特之间的换算依赖兰道尔极限，而兰道尔极限是理论下界，实际认知系统的信息处理效率远未达到该极限。统一为 bits 后，核算不再依赖热力学效率假设，也从根本上规避了“空转”——燃烧能量却不处理信息——的核算模糊性。

**Q2：净实焓为什么是减法（Γ = ΔΨ − ΔΠ）而不是除法（Ψ/Π）？**

A2：除法形式（Ψ/Π）衡量的是**效率**，减法形式（ΔΨ − ΔΠ）衡量的是**净增益**。0.2.x 版本中大量使用“净实焓为正”来指代 Ψ − Π > 0，同时又使用“Ψ/Π > 1”来指代效率比率，两者在自然语言中极易混淆。0.3.0 版本明确规定：净实焓仅指差值 Γ，效率比率称为实熵汇率 η。一个系统可以 η > 1（效率高）但 Γ ≈ 0（净增益微弱，如高度优化的空转），也可以 η < 1（效率低）但 Γ > 0（净增益为正，如公共图书馆的净实焓约为 +0.3x ~ +0.7x Π）。区分两者，是精确审计的前提。

**Q3：PTV 与熵价值论（ETV）如何对接？**

A3：ETV 描述了价值如何从稀缺性中产生（价值 = 稀缺性的熵度量），PTV 描述了价值如何在认知系统之间流动（剥削 = 熵差）。ETV 的核心公式是对数形式的价值函数，PTV 的核心公式是 Γ = ΔΨ − ΔΠ。对接后的完整图景是：Π（生产者燃烧的不确定性）是价值的原料，Ψ（消费者减少的不确定性）是价值的实现，Γ 是价值的净分配。剥削在 ETV 的框架中可被理解为熵差的截留。

**Q4：PTV 与 ETV 的主要区别是什么？**

A4：ETV 是**实证理论**，旨在解释和预测经济价值的决定因素（垄断、稀缺性、市场规模如何影响价格）。PTV 是**规范理论**，旨在审计价值的伦理分配（谁的不确定性被燃烧了？谁的不确定性减少了？差额被谁截留了？）。ETV 告诉你商品**值多少钱**；PTV 告诉你交易**是否公平**。两者互补而非竞争。

**Q5：如何测量 Ψ 和 Π？**

A5：直接以 bits 测量理想而困难。实践中采用**代理变量法**：Π 可通过算力消耗（FLOPs）、人力工时、注意力时长等可审计指标估算，并换算为信息处理比特的上限；Ψ 可通过消费者的行为改变量、决策质量提升、替代物的市场货币价格等代理指标反推。PTV 的核算精度取决于代理变量的选择，但方向性判断（Γ 的正负、剥削的存在）在多数情况下无需精确测量即可成立——正如判断“房间越来越热”无需温度计的绝对刻度。

**Q6：PTV 和传统成本效益分析有什么区别？**

A6：传统成本效益分析以货币为价值尺度，在公共物品（价格为 0）面前遭遇除零问题，只能依赖支付意愿问卷——其结果被富人的偏好绑架。PTV 以比特为价值尺度，通过代理变量（如替代物的市场价格）锚定 Ψ 的物理底价，不依赖支付意愿。当 PTV 使用货币代理时，货币不是“偏好权重的投票机”，而是“物理代价的翻译器”。（详见本文档“与边际效用论的关系”一节。）

**Q7：为什么 Γ 可以是负数？**

A7：Γ < 0 意味着系统的 Π 燃烧超过了其 Ψ 产出，在价值论意义上属于净耗散。这在认知系统中极为常见：刷三小时社交媒体后的 Γ < 0（注意力燃烧了，不确定性未减少）；行政空转的 Γ < 0（人力工时燃烧了，公共服务未改善）；意义通胀的 Γ < 0（信息洪流燃烧了 Π，边际 Ψ 趋零）。Γ < 0 不一定是“恶”，但它是不可持续的——长期 Γ < 0 的系统必须依赖外部 Π 输入（吸血）才能维持，否则将被选择压力淘汰。

**Q8：PTV 如何处理“百年无人查阅的孤本”这种长远分配问题？**

A8：保护孤本所燃烧的 Π，在未被查阅时表面上 ΔΨ = 0，Γ < 0。但 PTV 将保护行为视为持有认知期权（cognitive option）。期权的价值不取决于是否被行权，而取决于行权时的潜在收益乘以行权概率。只要期权未被宣告失效（孤本未被销毁），其期权费就是合理的 Π 支出。期权价值的评估可通过预测市场聚合分散信息来完成。详见本文档“理论后果”中“时间性与沉没成本谬误”一节。
